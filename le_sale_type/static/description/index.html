Custom Sale Types with Unique Sequence Numbers

This module allows you to create different sales types, each with its own unique sequence number. It is ideal for businesses that need to categorize sales into different types with their own numbering schemes.

Key Features:

  • Create multiple sales types (e.g., Retail, Wholesale, Online).
  • Each sales type has a unique sequence for numbering orders.
  • Automatically assign the correct sequence number when a sale is created.
  • Fully integrated with the Odoo Sales module.

After installing this module, you can configure the different sales types from the Sales configuration settings and define the sequence numbering for each type. The sequence numbers will automatically be applied when creating new sales orders.
