Form View Edit Button Restore

This module restores the classic Edit/Save workflow in Odoo 18 form views

  1. This module restores the classic Edit/Save workflow in Odoo 18 form views.
  2. Form views open in read-only mode by default.
  3. An Edit button is shown; Save and Discard are hidden.
